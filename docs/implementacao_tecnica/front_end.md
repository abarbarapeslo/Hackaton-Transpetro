# Frontend

## Interfaces da Plataforma Reevo

&emsp;This section presents the main screens of the Reevo application, covering both the **Farmer** and **Investor** profiles. Each screen is accompanied by its rationale, element description, and image insertion space.


&emsp;The front-end application can be viewed via this link:  [Reevo](https://Reevo-agro-hub.lovable.app/)

---

### Home (Agricultor e Investidor)

&emsp;The initial screen is a dashboard that presents the main platform information for both the farmer and the investor, allowing the user to choose their priority profile (**Farmer** or **Investor**). It displays a simplified initial dashboard with shortcuts to the main functionalities.

&emsp;Screen that presents the credit analysis result to the farmer in a **didactic manner**. It shows a **numerical score (0–1000)**, classification (A–E), an explanation of positive factors and areas for attention, in addition to **simulations for score improvement**.

## Conclusão

&emsp;The presented screens correspond to the already developed interfaces of the Reevo platform, comprehensively covering the journeys of the two main profiles: **Farmer** and **Investor**. Each screen was implemented based on **functional requirements**, ensuring **usability, clarity of information, and adherence to the defined visual identity**. The entire structure, from initial access to contract management and the marketplace, successfully unites **technology, security, and socio-environmental impact**.
